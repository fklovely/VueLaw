# 未成年人保护在线法律咨询服务平台

## 一、项目概述

### 1.1 项目简介
本项目是一个面向未成年人保护领域的在线法律咨询服务平台，旨在为用户提供专业的法律咨询、法律文书、案例检索、法规查询、在线课堂、互动问答等服务。

### 1.2 用户角色
| 角色 | 说明 |
|------|------|
| 客户(client) | 普通用户，可发起咨询、下载文书、学习课程等 |
| 律师(lawyer) | 提供法律咨询、文书服务、发布课程和文章 |
| 管理员(admin) | 系统管理，包括用户管理、内容审核、系统配置等 |

### 1.3 核心功能
- **律师服务**：律师列表、律师详情、在线咨询
- **法律文书**：文书模板下载、文书定制、文书审核
- **案例检索**：法律案例查询、案例详情
- **法规查询**：法律法规检索、法规解读
- **在线课堂**：法律课程学习、视频播放
- **互动问答**：法律问题提问、律师解答
- **普法文章**：法律知识普及文章

## 六、部署说明

### 6.1 环境要求
- JDK 1.8+
- MySQL 8.0+
- Node.js 18+
- Maven 3.6+

### 6.2 后端部署

#### 1. 创建数据库
```bash
mysql -u root -p
CREATE DATABASE legal_platform CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci;
USE legal_platform;
SOURCE database.sql;
```

#### 2. 修改配置
编辑 `backend/src/main/resources/application.yml`：
```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/legal_platform?useUnicode=true&characterEncoding=utf8&useSSL=false&serverTimezone=Asia/Shanghai
    username: root
    password: your_password
```

#### 3. 编译运行
```bash
cd backend
mvn clean package
java -jar target/legal-platform-0.0.1-SNAPSHOT.jar
```

### 6.3 前端部署

#### 1. 安装依赖
```bash
cd frontend
npm install
```

#### 2. 开发环境运行
```bash
npm run serve
```

#### 3. 生产环境构建
```bash
npm run build
```

构建产物在 `dist` 目录，部署到 Nginx 或其他 Web 服务器。

### 6.4 Nginx配置示例
```nginx
server {
    listen 80;
    server_name your-domain.com;
    
    location / {
        root /path/to/dist;
        index index.html;
        try_files $uri $uri/ /index.html;
    }
    
    location /api {
        proxy_pass http://localhost:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
    
    location /uploads {
        alias /path/to/uploads;
    }
}
```

## 八、开发说明

### 8.1 后端开发规范
- Controller 负责接口定义和参数校验
- Service 负责业务逻辑处理
- Mapper 负责数据库操作
- 使用 MyBatis-Plus 的 LambdaQueryWrapper 进行条件查询
- 统一使用 Result 类返回结果

### 8.2 前端开发规范
- 使用 Element UI 组件库
- API 请求统一放在 `src/api` 目录
- 页面组件放在 `src/views` 目录
- 公共组件放在 `src/components` 目录
- 使用 Vuex 管理全局状态

### 8.3 代码规范
- 遵循阿里巴巴 Java 开发手册
- 遵循 Vue 官方风格指南
- 代码注释使用中文
